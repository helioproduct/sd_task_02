# Домашнее задание 02: REST API сервиса

## Вариант 9. Электронная почта

Приложение работает с тремя сущностями:

- пользователь
- почтовая папка
- сообщение

Реализованные операции:

- создание пользователя
- вход и получение JWT-токена
- поиск пользователя по логину
- поиск пользователя по маске имени и фамилии
- создание новой папки
- получение списка папок пользователя
- создание письма в папке
- получение всех писем в папке
- получение письма по коду

Защищённые endpoints требуют заголовок `Authorization: Bearer <token>`.

Особенности реализации:

- один сервис на `userver`
- одна база `SQLite`
- порт `8080`
- Swagger UI доступен по `/swagger`
